# Lessons
- Always verify changes on the production URL (https://aicoach.epnstore.com.ar/).
- Fix bugs autonomously, requiring zero hand-holding.
- Demand elegance but keep things simple and minimal.
- UI Layout: Related form controls (like toggle switches and their corresponding inputs) should generally be grouped horizontally (e.g. `flex-row items-center gap-4`) rather than stacked vertically to save vertical space.
